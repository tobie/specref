Specref

An Open-Source, Community-Maintained Database of
Web Standards & Related References.

Search

Search by reference, title, keyword, editor, date or URL.

Use

Fetch normalized JSON data through the Web API or rely on existing integration in spec-authoring tools.

Contribute

Add new references and update existing ones on GitHub.

Automate

Standards body? Publisher? Get in touch for your list of references to be synchronized hourly.

Features

  • Over 23,000 references and counting.
  • Normalized JSON data.
  • Simple Web API.
  • Open-source and community-maintained.
  • Automatic update of W3C, WICG, WHATWG, WG21, IETF, and Unicode references.
  • Frequent updates.
  • Built into Bikeshed and ReSpec.
